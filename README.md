        <h2>About the Project</h2>
        <p>
            NanoGPT is a simple character-level language model project based on the Tiny Shakespeare dataset.
            It is inspired by Andrej Karpathy's work and demonstrates fundamental concepts of text generation
            using machine learning techniques.
        </p>

        <h2>Features</h2>
        <ul>
            <li>Downloads and preprocesses the Tiny Shakespeare dataset.</li>
            <li>Analyzes text data, including total characters and vocabulary size.</li>
            <li>Trains a character-level language model to generate Shakespeare-like text.</li>
        </ul>

        <h2>How to Use</h2>
        <ol>
            <li>Clone the project repository from GitHub.</li>
            <li>Install the required Python libraries.</li>
            <li>Run the Jupyter Notebook to execute the project.</li>
        </ol>

        <h2>Project Highlights</h2>
        <ul>
            <li>Dataset size: 1,115,394 characters.</li>
            <li>Vocabulary size: 65 unique characters.</li>
            <li>Generates realistic Shakespearean text using a trained model.</li>
        </ul>
